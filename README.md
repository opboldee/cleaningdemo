# Cleaning US Landing Page

A modern, responsive landing page for a residential and commercial cleaning company.

## Features
- Fully responsive (mobile, tablet, desktop)
- Smooth scroll navigation
- Interactive FAQ accordion
- Quote request modal
- Before/after gallery with hover effects
- 3 pricing packages with "Most Popular" badge
- Customer testimonials
- Trust features section

## Setup

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Build for production
npm run build
```

## Tech Stack
- React + TypeScript
- Vite
- Tailwind CSS
- Lucide React icons

## File Structure
```
cleaningwebsite/
├── index.html          # HTML entry point
├── src/
│   ├── App.tsx         # Main landing page component (all 10 sections)
│   ├── main.tsx        # React entry point
│   └── index.css       # Tailwind CSS with custom utilities
├── tailwind.config.js  # Tailwind configuration
├── postcss.config.js   # PostCSS configuration
└── vite.config.ts      # Vite configuration
```
